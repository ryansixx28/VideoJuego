# Requerimientos

## Funcionales
- Jugar quiz por niveles (CCNA, CCNP, CCIE)
- Autenticación con Google
- Carga de preguntas desde Firebase Firestore
- Ranking/Leaderboard por nivel
- Comodines (50/50, +15s, Saltar)

## No funcionales
- Desempeño: respuesta <100ms UI local
- Mantenibilidad: React/Vite, componentes desacoplados
- Seguridad: Reglas Firestore por usuario y solo lectura de preguntas
